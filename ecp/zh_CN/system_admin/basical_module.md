# 基础模块管理

ECP 分为**系统管理**和**工作台**两类界面：

- **系统管理**主要负责企业级管理、系统级设置等操作；
- **工作台**主要负责 EMQX 集群管理、边缘服务管理、监控告警、日志等服务管理和运维类的操作。

基础模块管理旨在帮助用户将初次安装部署的 ECP 进行组织、项目、用户、认证等方面的配置和使用介绍，完成基础模块的配置后，可以进行 ECP 在业务层的使用。
